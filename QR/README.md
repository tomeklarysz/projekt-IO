INSTALLATION

Install dependencies:
```
pip install -r requirements.txt
```


USAGE

Run the application:
```
python main.py
```

Available commands:
  - generate qr code  : Create a new QR code with unique ID
  - scan qr code      : Scan and verify QR code using camera
  - remove qr code    : Scan and remove QR code from database
  - exit              : Quit application
